# Setup Instructions

## Quick Start

1. **Install dependencies**

```bash
npm install
```

If you encounter peer dependency warnings, use:

```bash
npm install --legacy-peer-deps
```

2. **Set up environment variables**

```bash
cp .env.example .env
```

Edit `.env` and add your:

- Supabase URL and anon key (required)
- Google GenAI API key (optional)
- Agno AgentOS URL (optional, defaults to http://localhost:8000)

3. **Initialize Husky Git hooks**

```bash
npm run prepare
```

4. **Start development server**

```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## Troubleshooting

### Dependency Installation Issues

If you see peer dependency conflicts, try:

```bash
npm install --legacy-peer-deps
```

### TanStack Router Route Tree

The route tree (`src/routeTree.gen.ts`) is automatically generated by the TanStack Router Vite plugin. It will be created/updated when you run:

```bash
npm run dev
```

or

```bash
npm run build
```

### TypeScript Errors

Run type checking:

```bash
npm run type-check
```

### Linting Issues

Fix auto-fixable issues:

```bash
npm run lint:fix
```

### Missing Route Tree

If you see errors about missing `routeTree.gen.ts`, the Vite plugin will generate it automatically on first run. Make sure:

1. `@tanstack/router-vite-plugin` is in dependencies
2. The plugin is configured in `vite.config.ts`
3. You have route files in `src/routes/`
